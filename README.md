# FinEdge & Associates | Chartered Accountants Website

A premium, multi-page professional website template for Chartered Accountant firms, built with React, Vite, and GSAP.

- **Multi-Page Architecture**: Full routing system using `react-router-dom`.
- **Premium Design**: Modern, clean, and professional aesthetics with a forest green and gold theme.

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
